Welcome to E08014 Paper Preparation Pages.

I am setting up the the GitHub for everybody to work together and prepare the E08014 papers. (Note: it is still under development!!!)

How to set up githup:

Following the introduction on the official website: https://help.github.com/

How to link with your local directory

Method#1: Download the "tar ball" or "zip" files from the two buttons on the right panel. Method#2: For Linux or Mac, type the following commends in your terminal 

$ cd /your_path_to_store_the_whole_directory

Then type the following to download the entire directory

$ git clone git@github.com:goarsenal/e08014_papers.git

How to sync the files

Then to go the new created directory, and sync your "fork" with these files

git remote set-url upstream git@github.com:goarsenal/e08014_papers.git

Check whether you do correctly

$git remote -v

To keep it update:

$git fetch upstream
